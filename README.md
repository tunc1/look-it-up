# Look it Up

Search English Words on an Online Dictionary.

## How to Use

1. Select the word you would like to look up
2. Right click and click "Look it Up"
3. A new tab with the dictionary you set will be opened.

## Set Up
1. Find the url that takes you to the entry for a word in the dictionary you would like to use.
```
Example: https://dictionary.cambridge.org/dictionary/english/do
```
3. Replace the word with %searchText% in the url.
```
Example: https://dictionary.cambridge.org/dictionary/english/%searchText%
```
3. Paste this url on the pop up and click "Update URL"
